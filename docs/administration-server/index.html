管理サーバー

WebLogic Remote Consoleは、WebLogic管理サーバーに直接接続して、そのドメインを表示および編集します。

WebLogic Remote Consoleは、WebLogic Server RESTful管理サービスによって生成されたREST APIを介して管理サーバーに接続することによって、ドメインを管理します。 WebLogic RESTful管理サービスは、WebLogic Server MBeansからマップされたRESTリソースを動的に生成します。 WebLogic Remote Consoleは、これらのRESTリソースに基づいて、ドメインのMBean構造を反映し、リモートで実行されているドメインに接続できるグラフィカル・ユーザー・インタフェースを生成します。

詳細は、「RESTful管理サービスを使用したOracle WebLogic Serverの管理」の「WLS RESTful管理インタフェースについて」を参照してください。

WebLogic Serverドメインの概要、その内容およびドメイン構成に関する一般的なガイダンスは、「Oracle WebLogic Serverのドメイン構成について」の「Oracle WebLogic Serverドメインの理解」を参照してください。

WebLogic Remote Consoleは、WebLogic Serverドメインの管理に使用できるいくつかのツールの1つで、それぞれに独自の利点があります。 その他のシステム管理オプションのリストについては、「Oracle WebLogic Serverの理解」の「管理タスクに適したテクノロジの選択」を参照してください。
