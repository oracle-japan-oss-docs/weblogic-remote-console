WebLogic Remote Consoleのアップグレード

最新のWebLogic Remote Consoleにアップグレードし、コンソールとWebLogic Serverの両方の最新機能を利用します。

WebLogic Remote Console「拡張機能」は定期的に更新されます。 WebLogic Remote Consoleのアップグレード時に使用可能な拡張の新しいバージョンもあるかどうかを確認します。

拡張機能の以前のバージョンを削除して置き換えます。 拡張機能の複数のバージョンをインストールしたままにすると、予期しない動作が発生する可能性があります。

  1. 「ヘルプ」 > 「WebLogic Remote Console更新の確認」を開きます。

  2. 「ダウンロードしてインストール」を選択して、更新をすぐにダウンロードします。 更新はバックグラウンドで実行されるため、中断せずにWebLogic Remote Consoleで作業を続行できます。 次回コンソールを起動すると、更新が適用されます。

    「サイトに移動」を選択して、更新を手動でダウンロードできる「GitHubリポジトリ」に移動します。

Windows上で2.2.0以前からアップグレードする場合、以前のバージョンをアンインストールしようとするとアップグレードが失敗することがあります。 修正するには、PATH環境変数を編集して、%SystemRoot%\system32がfindコマンドを含む最初のエントリであることを確認してから、アップグレードを再試行します。
