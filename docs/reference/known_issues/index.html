既知の問題

このページには、WebLogic Remote Consoleの制限と既知の問題(および使用可能な場合はその回避策)が一覧表示されます。

制限事項

WebLogic Remote Consoleの現在のリリースでは、次のことはできません:

  • WebLogic Remote Consoleを使用してWebLogic Serverを構成中にWLSTスクリプトを記録します。
  • アプリケーションが正常にデプロイされた場合は、自動的にテストします。 手動オプションについては、「アプリケーション・デプロイメントのテスト」を参照してください。
  • WDTモデル・ファイル・プロバイダのセキュリティ・プロバイダの順序を変更します。 ただし、WDTモデル・ファイルをダウンロードし、YAMLファイルを手動で編集して、セキュリティ・プロバイダの順序を再配置できます。

制限されたMBeanプロパティのサポート

WebLogic Remote Consoleには、WebLogic Server管理コンソールでサポートされるMBeanプロパティの大部分が含まれます。 ただし、WebLogic REST APIの制限により、一部(非推奨のものを含む)が欠落しています。 WebLogic Remote Consoleで現在使用できないMBeanプロパティが必要な場合は、お知らせください。
